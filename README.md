# Tikkeul React Web

#### Install

```
yarn install
```

#### Dev mode

```
yarn dev
```

#### Storybook

```
yarn sb
```

#### 기타 커스텀 스크립트

```bash
yarn gen:icon {컴포넌트 이름}  
# SVG 아이콘 컴포넌트 템플릿 생성
# e.g. yarn gen:icon ArrowTop  

yarn gen:index {타겟경로}  
# 디렉토리 기준 배럴 파일 생성
# e.g. yarn gen:index src/components/icons
```

## 개발환경

- NodeJS-18.18.0
- yarn-1.22.21
